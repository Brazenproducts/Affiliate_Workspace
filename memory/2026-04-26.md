### Trending Positive Merch — POD Business Idea (6:39 PM UTC)

Mitch saw a garbage AI-generated t-shirt site (nowbestshirt.com) capitalizing on a news event with zero taste or context. Sparked an idea: do the same concept but ONLY positive/wholesome viral moments, with actual design quality.

**Concept:** Print-on-demand Shopify store that reacts fast to trending positive moments (sports, pop culture, wholesome viral, cultural events). No controversial/violent/political stuff.

**Model:**
- Shopify store + POD app (Printful, Printify, or Gooten)
- Zero inventory — POD handles print, pack, ship
- Typical margins: $8-12 base cost, sell $25-30
- Speed is everything — designs need to go live within hours of a moment trending
- Axl monitors trends, flags opportunities, generates designs, pushes to store

   - NowWearing
   - CatchTheMoment
   - BuzzWear
   - GoodNewsTees
   - MomentDrop
   - SnapMerch
   - Consider using an owned GoDaddy domain if one fits

2. **POD provider comparison needed:**
   - Printful — quality reputation, US/EU fulfillment, higher base cost
   - Printify — cheaper, multiple print providers, variable quality
   - Gooten — mid-range, good Shopify integration
   - Compare: base cost per tee/hoodie, print quality, shipping speed, Shopify app rating, design upload API (for automation)

3. **Automation pipeline:**
   - News/trend monitoring (RSS, X trending, Google Trends)
   - Axl flags positive-only opportunities
   - Design generation (text-based designs fastest, xAI/Grok for images ~$0.02/ea)
   - Auto-push to Shopify via API
   - Could potentially have designs live within 1-2 hours of a moment trending

4. **Legal considerations:**
   - Can't use real people's names/likenesses without rights (right of publicity)

   - "Inspired by" designs, not direct name/face usage
