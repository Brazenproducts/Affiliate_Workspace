## thedailycheer.com Zen Lofi Widget — Bug Hell (20:26–21:15 UTC)

### Root causes found and fixed:
1. **`<script src="app.js">` was missing from index.html** — literally no JS was running. Fixed ~20:33 UTC.
2. **Duplicate CSS blocks** — old `.chill-panel` definition overriding new styles. Removed ~182 lines of duplicate CSS.
3. **Stats counter** — IntersectionObserver threshold 0.3 never fired on mobile. Fixed to run on load + observe.
4. **iOS AudioContext** — `async/await` broke iOS tap-to-play requirement. iOS requires synchronous audio start.
5. **YouTube iframes** — some video IDs are embed-disabled ("video hasn't loaded" error). Replaced with verified IDs.
6. **`stopAll()` function** — nature audio and YouTube iframe were not being cleared when switching tabs/sounds. Fixed.
7. **Page zoom** — YouTube iframe load was triggering iOS viewport zoom. Fixed with `user-scalable=no` on viewport.
