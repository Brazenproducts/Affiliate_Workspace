# Moen A501 Nozzle — Product Photo Shot List

Amazon requires minimum 1 image, allows up to 9. For a strong listing, aim for 7.

## Required Photos

### 1. Main Image (WHITE BACKGROUND — Amazon requirement)
- Product only on pure white background (RGB 255,255,255)
- Show all 4 nozzles arranged neatly (slight overlap or fan pattern)
- Fill 85%+ of the frame
- No text, logos, or graphics on main image
- Minimum 1600px on longest side (for zoom)

### 2. Scale/Size Reference
- Single nozzle held between thumb and index finger
- Shows actual size clearly
- Clean background (white or light)

### 3. Compatibility Shot
- Nozzle next to a Moen spray head (A501 visible if possible)
- Arrow or callout showing where it goes
- Text overlay: "Fits Moen A501 Pulldown Spray Heads"

### 4. Problem/Solution — The Money Shot
- Side-by-side: degraded/cracked original rubber nozzle vs. new silicone replacement
- Text overlay: "Original Rubber (Fails)" vs "Upgraded Silicone (Lasts)"
- This is the image that sells the product — make the problem visceral

### 5. Installation Steps
- 3-panel infographic:
  - Panel 1: "Remove old nozzle" (hand pulling out degraded rubber)
  - Panel 2: "Insert new nozzle" (hand pushing in silicone replacement)
  - Panel 3: "Done! Like new" (faucet running with clean spray pattern)
- Text overlay: "60-Second Install — No Tools Required"

### 6. Before/After Spray Pattern
- Split image showing:
  - Left: uneven/weak spray from degraded nozzle
  - Right: clean, even spray from new silicone nozzle
- Taken at the kitchen sink with faucet running

### 7. What's In The Box
- Flat lay of all 4 nozzles with packaging
- Text overlay: "4-Pack — Keep Spares On Hand"

## Photo Tips
- Use natural light or a lightbox for the main image
- Phone camera is fine if it's a recent iPhone — just make sure focus is sharp
- Minimum 1600x1600px (2000x2000 preferred)
- Save as JPEG or PNG, sRGB color
- No watermarks on any image

## Optional Bonus Photos
- Lifestyle shot: someone using the faucet normally in a real kitchen
- Close-up of silicone material texture (shows quality)
- Packaging shot if you have branded packaging
